# Video séquence

### Installation

`npm install`

### Utilisation

Le projet teste l'idée d'enchaîner des vidéos les unes après les autres en cliquant.

Pour changer les vidéos dans `components/VideoSequence.jsx` :

````
   const videos = [
        '/video1.mp4',
        '/video2.mp4',
        '/video3.mp4'
    ];
````
